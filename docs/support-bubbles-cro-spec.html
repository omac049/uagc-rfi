What Are Support Bubbles?

Support bubbles are a contextual help system designed to guide users through form completion, reduce form abandonment, provide just-in-time assistance, and encourage users who are struggling.

⚠️ Engagement Gate

Help messages ONLY appear AFTER the user has engaged with the form (first focus or click on any form field). This prevents intrusive pop-ups on page load and ensures a non-disruptive experience.

JavaScript Implementation

Core functions and initialization logic
1. showSmartTip() - Display Tips 
function showSmartTip(message, title = 'Tip', icon = '💡', type = 'info') {
  const tipContainer = document.getElementById('smartTipContainer');
  if (!tipContainer || !userEngaged) return;
  tipContainer.className = `smart-tip-container ${type}`;
  tipContainer.innerHTML = `...`;
  tipContainer.style.display = 'block';
  if (type !== 'warning') {
    setTimeout(() => tipContainer.style.display = 'none', 10000);
  }
}
2. showContextualHelp() - Help Modal 
function showContextualHelp() {
  const helpContent = getContextualHelpContent();
  const modal = document.createElement('div');
  modal.innerHTML = `<div class="help-modal-overlay">...</div>`;
  document.body.appendChild(modal);
  trackEvent('help_modal_opened', { step: currentStep });
}
3. setupEngagementDetection() - User Tracking 
function setupEngagementDetection() {
  const formFields = form.querySelectorAll('input, select, textarea');
  const handleFirstInteraction = () => {
    if (!userEngaged) {
      userEngaged = true;
      setTimeout(() => showSmartTip('Welcome!...', '👋 Welcome!', '👋', 'info'), 1000);
      setupActivityTracking();
    }
  };
  formFields.forEach(f => {
    f.addEventListener('focus', handleFirstInteraction, { once: true });
    f.addEventListener('click', handleFirstInteraction, { once: true });
  });
}
4. setupActivityTracking() - Abandonment 
function setupActivityTracking() {
  let inactivityTimer;
  const resetTimer = () => {
    clearTimeout(inactivityTimer);
    inactivityTimer = setTimeout(offerAbandonmentHelp, 60000);
  };
  document.addEventListener('mousemove', resetTimer);
  document.addEventListener('keypress', resetTimer);
  resetTimer();
}

function initTooltips() {

    }
  });
}

QA Checklist

Quality assurance verification items

QA Checklist

  • Smart tip container appears on all defined triggers
  • Smart tip auto-dismisses after 10 seconds (info/success types)
  • Warning tips persist until manually closed by user
  • Floating help button visible and clickable on all pages
  • Help badge appears after 60 seconds of inactivity
  • Help modal opens/closes correctly (overlay, × button, Escape)
  • Help modal content switches based on current form step
  • Tooltips work on hover (desktop) and click (mobile)
  • All events fire correctly to GA4
  • Mobile responsive design works (test 320px-768px)
  • Accessibility: all focus states, ARIA labels present
  • No JavaScript errors in browser console
  • Performance: no visible lag, jank, or layout shift
  • Print stylesheet hides all support bubble elements
  • Form still submits correctly with support bubbles active
